2  Notes

This document lists UCI datasets that are potentially useful for regression projects.
For each dataset below, I include:

  • the UCI webpage link,
  • an R code chunk to import the dataset,
  • basic information such as number of variables, number of observations, response variable, and topic.

Most import examples use the {ucimlrepo} package, which can fetch datasets directly from the UCI Machine Learning Repository from within R.

# install.packages("ucimlrepo")
library(ucimlrepo)
